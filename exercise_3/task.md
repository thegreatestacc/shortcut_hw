# Практическое задание: воркер с таймаутом и остановкой

## Задача

Реализуй воркер, который:

1. В бесконечном цикле читает задания из канала jobs, обрабатывает каждое задание (например, time.Sleep(500ms) + вывод).
2. При этом воркер должен останавливаться по сигналу stopCh (канал struct{}), прерывая обработку задания по таймауту (например, 1 секунда) через select.

---

## Структура цикла. Внутри обработки:

1. select между:
   time.After(timeout) — считаем, что задание "не успело";
   stopCh — немедленный выход;
2. завершение обработки (можно смоделировать через второй канал или просто time.Sleep + ещё один select).

---
## Требования:
```main```
1. Запускает воркера(ов).
2. Отправляет несколько заданий.
3. Через некоторое время посылает stop (закрывает stopCh).
4. Корректно завершает программу.

```worker```
1. Уважает сигнал об остановке.
2. Корректно завершает работу.
